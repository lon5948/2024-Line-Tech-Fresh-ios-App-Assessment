# 2024-Line-Tech-Fresh-ios-App-Assessment

Use the API from the [Taiwan Government Open Data platform](https://data.gov.tw) to retrieve information about significant international epidemics and Taiwan's National Health Insurance contracted medical institutions (including pharmacies, clinics, district hospitals, regional hospitals, and medical centers).

## Datasets

- [International Epidemic Information](https://data.gov.tw/dataset/10568)
- [Pharmacy](https://data.gov.tw/dataset/39284)
- [Clinic](https://data.gov.tw/dataset/39283)
- [District Hospital](https://data.gov.tw/dataset/39282)
- [Regional Hospital](https://data.gov.tw/dataset/39281)
- [Medical Center](https://data.gov.tw/dataset/39280)
